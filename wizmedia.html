Wizmedia

Description

A graphic design portfolio made for Wizmeida. Inspired by Alex Coven portfolio design, I worked with Wizmeida to create his web portfolio. It has a collection of all his work ranging from logos he had created to brands he created as well. Feel free to reach out to him for graphic design work. Wizmedia is a graphic designer based in London, England. He also happens to be my amazing older brother.

Motivation

Wizmedia was looking to create his web portfolio and wanted it to be a custom design. I felt it would be a great way to go back to the basics of writing plane html, css and javascript/jqurey creating webpages from wireframes.

Lessons Learnt

Refereshed my knowledge on basic web languages and how to actually created a website without using a framework. Learnt how to make the web pages accessible and viewer friendly to different devices. Also refreshed my knowledge on hosting a website on a server and working with custom domains.
